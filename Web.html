How to View HTML Source?

Have you ever seen a Web page and wondered "Hey! How did they do that?"

View HTML Source Code:

Right-click in an HTML page and select "View Page Source"(in Chrome) or "View Source" (in Edge), or similar in other browsers. This will open a window containig
the html source code of the page.

Inspect an HTML Element:

Right-click on an element (Or a blank area), and choose "Inspect" or "Inspect Elemnet" to see what elements are made up of (you will see both the HTML and the
CSS). You can also edit the HTML or CSS on-the-fly in the Element or Styles panel that opens.
